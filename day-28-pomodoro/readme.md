## Pomodoro
* A GUI-based Pomodoro app

### What does it do?
The Pomodoro will:
* Ask the user to click the Tomato image to start the timer


### How does it work?
On the command-line (or terminal), type the following:<br>
* python3 <b>pomodoro.py</b>

<b>NOTE:</b> It is assumed that the <b>Python 3 interpreter</b> had already been installed on your computer.
